This is a task From Magnet Brains where I have to Integrate Cart System and Payment Getway. 
To run this project you have -- 
1. Open project in VS code.
2. Press ctrl+shift+` to open terminal.
3. Then in terminal run ' cd backend ' and then ' npm i ' .
4. Now your Server dependencies are all downloaded now you have to start the server, for that run ' nodemon ' in terminal.
5. If everything is correct server will start.
6. Now to Start frontend, open new terminal and run ' cd frontend ' and then ' npm i ' .
7. Now your frontend dependencies are all downloaded now you have to start the frontend, for that run ' npm run dev ' in terminal.
8. If everything is correct you will see a link, you have to ctrt+rght click on it.
9. If both frontend and backend are correct you will see Home page.
